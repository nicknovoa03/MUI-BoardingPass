# [Minting dApp for Black Box Collective](https://blackboxcollective.io/)


## Built with

- React
- MUI
- React Scripts
- web3.js


## Quick start with React Scripts

- Install dependencies: `yarn install`
- Start the server: `yarn run start`
- Build on production: `yarn run build`
